# dasboot
##### A Tiny x86 Bootloader

To compile the code, type
```
nasm -f bin boot.asm -o boot.com
```

To emulate the boot loader, type
```
bochs -f bochsrc.txt
```

Booty, booty, booty, booty, rockin' everywhere.
